1. **剩余收益模型（Residual Income Model）** - 核心模型

3. **相对估值法** - 市场比较

## 1. 剩余收益模型

### 模型公式

```
V0 = B0 + Σ(RIt / (1+r)^t) + TV / (1+r)^n

其中：
B0 = 账面价值
RI = 剩余收益 = Net Income - (r × B)
r = 权益资本成本
TV = 终值
```
